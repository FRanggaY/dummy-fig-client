## Step by Step

<br />

## 1. update package list

<CodeBlock children={`sudo apt update`}/>

## 2. install required package

<CodeBlock children={`sudo apt install apt-transport-https ca-certificates curl software-properties-common`}/>

## 3. GPG key for Docker

<CodeBlock children={`curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg`}/>

## 4. Add Docker Repository

<CodeBlock children={`echo "deb [arch=amd64 signed-by=/usr/share/keyrings/docker-archive-keyring.gpg] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list > /dev/null`}/>

## 5. Update package list again

<CodeBlock children={`sudo apt update`}/>


## 6. Install Docker

<CodeBlock children={`sudo apt install docker-ce`}/>


## 7. Install Docker Compose

<CodeBlock children={`sudo curl -L "https://github.com/docker/compose/releases/latest/download/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose`}/>

## 8. Allow Access

<CodeBlock children={`sudo chmod +x /usr/local/bin/docker-compose`}/>

## 9. Install gnome keyring

<CodeBlock children={`sudo apt-get install gnome-keyring`}/>

Congratulations. all setup already installed, check the result

## Check Docker Compose version

<CodeBlock children={`docker-compose --version`}/>

<br/>

## Check Docker version

<CodeBlock children={`docker --version`}/>
