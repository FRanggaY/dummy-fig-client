## Solusi
Untuk memperbaikinya, lakukan dua hal berikut:

## 1. Tingkatkan Batas Unggah:
Sesuaikan pengaturan server Anda (misalnya, Nginx atau Apache) untuk mengizinkan unggahan file yang lebih besar.

## 2. Periksa WAF (Firewall Aplikasi Web):
Pastikan Firewall Aplikasi Web Anda tidak memblokir unggahan yang sah. Jika perlu, hubungi dukungan AWS untuk menyesuaikan pengaturannya.

Dengan mengikuti langkah-langkah sederhana ini, Anda dapat mengatasi kesalahan "forbidden" dan memungkinkan unggahan file besar di situs web Anda.
